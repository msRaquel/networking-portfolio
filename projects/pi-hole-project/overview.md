## DNS Raspberry Pi-Hole (Sinkhole)

What is DNS? Domain Naming System - allows names like github.com to be mapped to ip addresses.We use DNS names because IP addresses aren't readable and when you want to search for github.com you probably don't want to type in 188.12.300.14 everytime (we remember names easier than number sequences).
